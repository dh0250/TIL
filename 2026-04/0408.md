# Java Script

## 1. data type

- 변수 : 데이터 저장할 때 쓰이는 저장소
- 키워드 변수명;

```jsx
// var: ES5 까지 사용했던 유일한 키워드
var number;
var number = 5;

// let: 값을 변경할 수 있는 변수 선언
let greeting;
greeting = "Hello, Node.js!";
console.log(greeting);

greeting = "Welcome to JavaScript!";
console.log(greeting);

// const: 재할당이 금지
//const num; // const 키워드로 선언한 변수는 반드시 선언과 동시에 초기화 해야 한다.
const num = 1;
```

- 변수 선언 규칙과 스타일
    - 변수 이름은 알파벳, 숫자, _, $만 사용할 수 있다.
    - 변수 이름은 숫자로 시작할 수 없다.
    - 카멜케이스(camelCase)를 사용하는 것이 일반적이다.
    - 예약어 사용 금지
    - 변수 이름에서 대소문자를 구분한다.
    - 의미 있는 이름 사용

## 2. Primitive types

```jsx
// 1. 숫자(number)
let age = 30;
console.log(age);

// 숫자 타입은 모두 실수로 처리되므로 둘은 같다.
console.log(10 === 10.0); //true
console.log(10 / 4); //2.5

// 2. 문자열
// 작은 따옴표(''), 큰 따옴표(""), 또는 백틱(``)으로 텍스트를 감싼다.
let string = "문자열";
console.log(string);

// 3. 불린(boolean) : 논리적 참, 거짓을 나타내는 true와 false 뿐이다.
let isStudent = true;
console.log(isStudent);

// 4. null: 명시적으로 값이 없음을 의도적으로 명시할 때 사용.
let address = null;
console.log(address);

// 5. undefined: 변수에 값이 할당되지 않았을 때 자동으로 할당되는 값.
let salary;
console.log(salary);

// 5. typeof : 데이터 타입을 확인할 때 사용
console.log(typeof age);
console.log(typeof string);
console.log(typeof isStudent);
console.log(typeof address); 
console.log(typeof salary);
```

## 3. 템플릿 리터럴

- ES6부터 도입 된 멀티라인 문자열, 표현식 삽입 등의 편리한 문자열 처리 기능을 제공하는 문자열 표기법
- 작은 따옴표, 큰 따옴표 대신 백틱(``)을 사용해 표현

```jsx
// 일반 문자열 내에서는 줄바꿈 허용되지 않음
// let str = "안녕하세요.
// 반갑습니다.";

let str = '안녕하세요.\n반갑습니다.';
console.log(str);

// 백틱을 사용하면 줄바꿈과 공백이 있는 그대로 적용됨
let multiline = `안녕하세요.
반갑습니다.`;
console.log(multiline);

let lastName = '홍'
let firstName = '길동';

console.log('제 이름은 ' + lastName + firstName + ' 입니다');

// 표현식 삽입 : ${}
console.log(`제 이름은 ${lastName}${firstName} 입니다.`);
```

## 4. 암묵적 타입 변환

- 문자열 타입으로 변환

```jsx
console.log(10 + '20'); //1020

console.log(1 + ''); 
console.log(NaN + ''); //not a number (NaN)
console.log(undefined + ''); //undefined
```

- 숫자 타입으로 변환

```jsx
console.log(10 - '5'); // 5
console.log(10 * '5'); // 50
console.log(10 / '5'); //2
console.log(10 % 'JavaScript');  //NaN

console.log(10 > '5'); //true
```

- + 단항 연산자 : 피연산자가 숫자 타입의 값이 아니면 숫자 타입으로 암묵적 타입 변환 수행

```jsx
console.log(+''); // 빈문자열은 0으로
console.log(+true); // 1
console.log(+false); // 0
```

- 불리언 타입으로 변환
    - Falsy 한 값
        - false
        - 0 (숫자 0)
        - ‘’ (빈 문자열)
        - null
        - undefined
        - NaN (Not a Number)

```jsx
if(10 > 5) console.log("참 입니다.");
if('') console.log("if('')");
if(0) console.log("if(0)");
```

## 5. 명시적 타입 변환

- 문자열 타입으로 변환
    - String 생성자 함수 new 연산자 없이 호출
    
    ```jsx
    console.log(String(10));      // "10"
    console.log(String(true));    // "true"
    console.log(String(null));    // "null"
    ```
    
    - toString 메서드 사용
    
    ```jsx
    console.log((10).toString());
    ```
    
- 숫자 타입으로 변환
    - Number 생성자 함수를 new 연산자 없이 호출
    
    ```jsx
    console.log(Number('10.01'));  // 10.01
    console.log(Number('10원'));   // NaN
    console.log(Number(true));     // 1
    console.log(Number(false));    // 0
    ```
    
    - parseInt, parseFloat 함수 이용 (문자열 -> 숫자만 가능)
    
    ```jsx
    console.log(parseInt('10.01'));   // 10
    console.log(parseFloat('10.01')); // 10.01
    ```
    
- 논리 타입으로 변환
    - boolean 생성자 함수를 new 연산자 없이 호출하는 방법
    
    ```jsx
    console.log(Boolean("JS")); // true
    console.log(Boolean(0));    // false
    console.log(Boolean(NaN));  // false
    ```
    
    - ! 부정 논리 연산자를 두 번 사용하는 방법
    
    ```jsx
    console.log(!true);   // false
    
    console.log(!!'');    // false //boolean으로 바꿔줌
    console.log(!!1);     // true
    console.log(!!0);     // false
    ```

# ✏️Operator

## 1. 연산자

1. 산술 연산자

```jsx
let a = 10;
let b = 3;

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(a / b);
console.log(a % b);
console.log(a ** b); // 거듭제곱
```

1. 할당 연산자

```jsx
let c = 5;
c += 3;
console.log(c); //8

c -= 2; // c = c - 2
console.log(c); //6
```

1. 증감 연산자

```jsx
let d = 5;
console.log(d++);  // 후위 증가
console.log(++d);  // 전위 증가
console.log(d--);  // 후위 감소
console.log(d);
```

---

## 2. 비교 연산자

1. 동등 연산자 (==, ≠) : 먼저 암묵적 타입 변환을 통해 타입을 일치시킨 후 같은 값인지 비교

```jsx
let x = 10;
let y = 20;

console.log(x == y);     // == : 값이 같으면 true  //출력 : false
console.log(x != y)      // != : 값이 다르면 true  //출력 : true
console.log(x == '10');  // 암묵적 타입 변환을 했기 때문에 true 반환
```

1. 일치 연산자 (===, ≠=) : 타입과 값이 모두 일치하는지 비교

```
console.log(x === 10);   // === : 값이 타입이 같으면 true
console.log(x === '10'); // false
```

1. 대소 비교 연산자

```jsx
console.log(x > y);
console.log(x <= y);
```

1. 문자열도 비교 가능(아스키코드 순서)

```jsx
console.log(`'cat' >= 'Zoo' : ${'cat' >= 'Zoo'}`); //true
```

## 3. 논리 연산자

1. 논리 AND (&&): 모두 true일 때만 true 반환

```jsx
const a = true;
const b = false;

console.log(a && a);
console.log(a && b);
```

1. 논리 OR (||): 둘 중 하나만 true여도 true 반환

```jsx
console.log(a || b);
```

1. 논리 NOT (!): true이면 false로, false이면 true로 반전

```jsx
console.log(!a);
console.log(!b);
```

- 표현식 평가하는 도중 평가 결과가 확정 된 경우 나머지 평과 과정을 생략하는 것

```jsx
console.log('apple' || 'banana'); // apple
console.log(false || 'banana');  // banana

console.log(false && 'banana') // false
```

- 단축평가를 이용한 null 체크!

```jsx
let obj = null;

// let objValue = obj.value; // TypeError: Cannot read properties of null (reading 'value')

let objValue = obj && obj.value; //false

/** 치명적 단점
 * 0이나 빈 문자열('')처럼 나름 의미가 있는 값들까지 '문제(Falsy)'로 취급
 */
```

- Optional Chaining (옵셔널 체이닝)

```jsx
const obj1 = null;

// obj가 null 또는 undefined 인 경우 undefined 반환
// obj가 다른 Falsy한 값(0, '')인 경우 그대로 0 또는 '' 반환
const objValue1 = obj?.value;
```

- null 병합 연산자 (??)

```jsx
let test = null ?? '기본 값';
console.log(test);

let value = '' ?? '기본 값';
console.log(value);
```

## 4. 삼항 연산자

let result = (age >= 20) ? "성인" : "미성년자";

## 5. 조건문

- 조건문 if

```jsx
let temperature = 5;

if (temperature < 10) {
    console.log("오늘은 춥습니다.");
} else {
    console.log("오늘은 날씨가 괜찮습니다.");
}
```

- 여러 조건을 처리할 때

```jsx
let score = 85;

if(score >= 90) {
    console.log("매우 우수");
} else if (score >= 70) {
    console.log("양호");
} else if (score >= 50) {
    console.log("보통");
} else {
    console.log("부족");
}
```

- 조건문 while

```jsx
let fruit = '바나나';

switch (fruit) {
    case '사과' :
        console.log('선택한 과일은 사과입니다.');
        break;
    case '바나나' :
        console.log('선택한 과일은 바나나입니다.');
        break;
    default :
        console.log('알 수 없는 과일입니다.');
}
```

## 6. 반복문

- 반복문 for

```jsx
for (let i = 1; i <= 5; i++) {
    console.log(`반복 횟수: ${i}`);
}

/**
 * 1부터 10까지의 숫자 중 짝수만 출력
 */
for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        console.log(i);
    }
}          
```
