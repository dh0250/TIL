## 03. scanner (스캐너)

콘솔로부터 사용자의 입력을 받기 위해 제공되는 클래스

---

✔ Scanner 기본 사용법

```java
import java.util.Scanner;

Scanner sc = new Scanner(System.in);

// 자료형별 입력 받기
String name = sc.nextLine();     // 한 줄 전체 (문자열)

int age = sc.nextInt();          // 정수
double height = sc.nextDouble(); // 실수

// 문자 입력 받기

```


✔ 안정적인 입력 (nextLine 통일)

# 01. conditional (조건문)

프로그램의 흐름을 조건에 따라 제어하며, 특정 조건 충족 시에만 코드가 실행되도록 함

---

✔ if, if-else, if-else-if 문

```java
int num = 10;

// 단일 if문
if (num > 0) {
    System.out.println("양수입니다.");
}

// if-else문
if (num % 2 == 0) {
    System.out.println("짝수입니다.");
} else {
    System.out.println("홀수입니다.");
}

// if-else-if문 (다중 조건)
int score = 85;
if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C 이하");
}
```

---

✔ switch 문

```java
char op = '+';
int first = 10, second = 20, result = 0;

switch (op) {
    case '+' : result = first + second; break;
    case '-' : result = first - second; break;
    default: System.out.println("잘못된 연산자"); return;
}

// Java 12+ 향상된 switch문 (값 반환 가능)
int modernResult = switch (op) {
    case '+' -> first + second;
    case '-' -> first - second;
    default -> 0;
};
```

# 02. looping (반복문)

특정 실행 코드 블록을 조건이 만족하는 동안 반복해서 실행함

---

✔ for 문

```java
// 1부터 10까지의 합계 구하기
int sum = 0;
for (int i = 1; i <= 10; i++) {
    sum += i;
}

// 중첩 for문 (구구단 예시)
for (int dan = 2; dan <= 9; dan++) {
    for (int su = 1; su <= 9; su++) {
        System.out.println(dan + " * " + su + " = " + (dan * su));
    }
}
```

---

✔ while 및 do-while 문

```java
// while문: 조건식 선검사 후 실행
int i = 1;
while (i <= 5) {
    System.out.println(i + "번째 출력");
    i++;
}

// do-while문: 최소 한 번은 무조건 실행
int count = 1;
do {
    System.out.println("일단 한 번은 실행됨");
} while (count < 0);
```

# 03. branching (분기문)

반복문의 흐름을 강제로 중단하거나 건너뛰기 위해 사용함

---

✔ break 문

```java
// 가장 가까운 반복문을 즉시 탈출
while (true) {
    if (condition) {
        break;
    }
}

// 중첩 반복문 탈출 (Flag 변수 활용 권장)
boolean isBreak = false;
for (int i = 0; i < 10; i++) {
    for (int j = 0; j < 10; j++) {
        if (i == 3 && j == 5) {
            isBreak = true;
            break;
        }
    }
    if (isBreak) break;
}
```

---

✔ continue 문

```java
// 현재 회차를 건너뛰고 다음 반복으로 이동
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue; // 짝수는 건너뜀
    }
    System.out.println(i); // 홀수만 출력
}
```

## ✔ Test1.java (성적 합격/불합격 판별)

```java
import java.util.Scanner;

public class Test1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int[] scores = new int[3];

        System.out.print("국어 점수를 입력하세요 : ");
        scores[0] = sc.nextInt();

        System.out.print("영어 점수를 입력하세요 : ");
        scores[1] = sc.nextInt();

        System.out.print("수학 점수를 입력하세요 : ");
        scores[2] = sc.nextInt();

        int sum = scores[0] + scores[1] + scores[2];
        double avg = (double) sum / 3;

        if (avg < 60) System.out.println("평균 점수 미달입니다.");
        if (scores[0] < 40) System.out.println("국어 점수 미달입니다.");
        if (scores[1] < 40) System.out.println("영어 점수 미달입니다.");
        if (scores[2] < 40) System.out.println("수학 점수 미달입니다.");

        if (avg >= 60 && scores[0] >= 40 && scores[1] >= 40 && scores[2] >= 40) {
            System.out.println("합격입니다.");
        }
    }
}
```

## ✔ Test4.java (Up-Down 숫자 맞추기)

```java
import java.util.Scanner;

public class Test4 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int random = (int)(Math.random() * 100) + 1;
        int count = 0;

        while (true) {
            System.out.print("정수를 입력하세요 : ");
            int num = sc.nextInt();
            count++;

            if (num > random) {
                System.out.println("입력하신 정수보다 작습니다.");
            } else if (num < random) {
                System.out.println("입력하신 정수보다 큽니다.");
            } else {
                System.out.println("정답입니다. " + count + "회만에 맞추셨습니다.");
                return;
            }
        }
    }
}
```
