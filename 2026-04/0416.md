# 🔗React

- Virtual DOM 을 사용하여 빠르고 효율적인 UI 렌더링을 제공(성능 향상)
- Component 를 사용하여 UI를 구성하며 상태(state) 및 속성(props) 을 활용하여 컴포넌트를 렌더링  (코드 재사용 및 개발 생산성 향상)
- 유연하고 확장 가능한 라이브러리

### 1. React 주요 요소

- 엘리먼트(Element)
- 컴포넌트(Component) -클래스형, 함수형
- Virtual DOM
- JSX - React에서 UI를 구성하는 데 사용하는 자바스크립트의 확장 문법

## 📝JSX

- jsx 안에서 사용할 자바스크립트 데이터와 함수

```jsx
const user = {
            name: '홍길동',
            age: 20
        };
        
        function introduce(user) {
            return `안녕하세요, 저는 ${user.name}이고 ${user.age}세 입니다.`;
        };

        function handleClick() {
            alert('버튼이 클릭되었습니다!');
        }

        const customStyle = {
            backgroundColor: 'cornflowerBlue',
            color: 'white',
            padding: '10px',
            borderRadius: '5px'
        }
```

- 컴포넌트

```jsx
function JsxComponent() {
            return (
                <>
                    <h2>1. { } 안에 자바스크립트 표현식을 넣을 수 있다. </h2>
                    <h3>이름: {user.name}</h3>
                    <p>소개: {introduce(user)}</p>

                    <h2>2. HTML 속성과 약간 다른 규칙을 따른다.</h2>
                    {/* class는 className으로, onclick은 onClick으로 (camelCase) */}
                    <button className="my-button" onClick={handleClick}>
                        클릭해보세요
                    </button>

                    <h2>3. 인라인 스타일은 '객체' 형태로 전달해야 한다.</h2>
                    <p style={customStyle}>
                        이 문단에는 커스텀 스타일을 적용하였다    
                    </p>

                    <h2>4. 모든 태크는 반드시 닫혀야 한다.</h2>
                    한 줄짜리 태그도 예외는 없다. <br /> <hr /> <input />
                </>
            )
        }

        ReactDOM.createRoot(document.getElementById('root')).render(<JsxComponent />);
```

### 2. MessageManager 컴포넌트

- 상태 2개 관리
    - `message`: 화면에 표시할 텍스트
    - `textColor`: 텍스트 색상
- 이벤트에 따라 상태 변경
    - 입장 버튼 → 환영 메시지 + 파란색
    - 퇴장 버튼 → 작별 메시지 + 빨간색

**포인트**: 상태 변경 시 컴포넌트 재렌더링

---

### 3. Counter 컴포넌트

- `number` 상태로 숫자 관리

### 잘못된 방식

```jsx
setNumber(number + 1);
setNumber(number + 1);
```

- 같은 값 기준으로 업데이트 → +1만 적용됨

### 올바른 방식 (함수형 업데이트)

```jsx
setNumber(prev => prev + 1);
setNumber(prev => prev + 1);
```

- 최신 상태 기준으로 계산 → +2 정상 적용

---

### 4. 핵심 포인트 정리

- 상태 업데이트는 **비동기적**
- 같은 상태를 여러 번 바꿀 때는 **함수형 업데이트 사용**
- 상태가 바뀌면 **컴포넌트는 다시 렌더링됨**

```jsx
    <div id="root"></div>
    
    <script type="text/babel">
        
        console.log(React);

        const {useState} = React;

        function MessageManager() {
            // [상태 값, 상태 변경 함수] = useState(초기값)
            const [message, setMessage] = useState('초기 상태 메세지입니다.');
            const [textColor, setTextColor] = useState('black');

            console.log('MessageManager 컴포넌트가 렌더링 되었습니다.');

            const handleEnter = () => {
                setMessage('안녕하세요! 환영합니다.');
                setTextColor('cornflowerblue');
            }
            const handleLeave = () => {
                setMessage('안녕히 가세요! 다음에 또 만나요.');
                setTextColor('tomato');
            }
            return (
                <div>
                    <h1>useState</h1>
                    <h2 style={{ color: textColor }}>{message}</h2>
                    
                    <button onClick={handleEnter}>입장</button>
                    <button onClick={handleLeave}>퇴장</button>
                </div>
            )
        }

        function Counter() {
            const [number, setNumber] = useState(0);

            const increaseByTwo = () => {
                setNumber((preveNumber) => preveNumber + 1);
                setNumber((preveNumber) => preveNumber + 1);
            }

            return (
                <div>
                    <h1>카운터</h1>
                    <h2>현재 숫자: {number}</h2>
                    <button onClick={() => {
                        setNumber(number + 1);
                        setNumber(number + 1);
                    }}>+2(동작 안함)</button>
                    <button onClick={() => setNumber(number - 1)}>-1</button>
                    <button onClick={increaseByTwo}>+2</button>
                </div>
            )
        }

        function App() {
            return (
                <>
                    <MessageManager />
                    <hr />
                    <Counter />
                </>
            )
        }

        ReactDOM.createRoot(document.getElementById('root')).render(<App />);
        
    </script>
```

### React 이벤트 처리 + form 관리 정리

1. 기본 form 처리 (SignupFormBasic)
- 각각의 input을 개별 state로 관리
- username, email을 따로 useState로 생성
- onChange에서 바로 상태 업데이트

핵심 흐름

입력 → onChange → 상태 변경 → 화면 반영

단점

input이 많아질수록 state와 코드가 증가

---

1. 객체로 form 관리 (SignupFormAdvanced)
- 관련 상태를 하나의 객체(form)로 묶어서 관리
- useState({ username, email })

핵심 로직

- name 속성을 활용해서 어떤 input인지 구분
- 기존 state를 복사한 후 필요한 값만 수정

동작 방식

- e.target에서 name, value 추출
- form 객체에서 해당 key만 업데이트

장점

- input이 많아져도 코드 재사용 가능
- 유지보수 훨씬 쉬움

주의

- 반드시 기존 state를 펼쳐서 복사해야 함 (불변성 유지)

---

1. 이벤트 함수 전달 (CustomButton)
- 부모 컴포넌트에서 만든 함수를 props로 전달
- 자식 컴포넌트에서 onClick에 연결

구조

- App → 함수 정의
- CustomButton → props로 받아서 실행

의미

- 컴포넌트 재사용성과 유연성 증가

---

1. 핵심 정리
- 간단한 form → 개별 state 사용
- 복잡한 form → 객체로 묶어서 관리
- input name 속성 활용하면 효율적
- 이벤트 함수는 props로 전달 가능

```jsx
<div id="root"></div>
    
    <script type="text/babel">

        const {useState} = React;

        function SignupFormBasic() {

            const [username, setUsername] = useState('');
            const [email, setEmail] = useState('');

            const handleSignup = () => {
                alert(`아이디:${username} 이메일: ${email} 회원가입 완료!`);
                setUsername('');
                setEmail('');
            }

            return (
                <div>
                    <h2>기본적인 이벤트 처리</h2>
                    <input 
                        type="text"
                        placeholder='아이디'
                        value={username}
                        onChange={e => setUsername(e.target.value)}
                    />    
                    <input 
                        type="email"
                        placeholder='이메일'
                        value={email}
                        onChange={e => setEmail(e.target.value)}
                    />   
                    <button onClick={handleSignup}>가입하기</button>
                </div>
            )
        }

        function SignupFormAdvanced() {
            // 관련된 상태들을 하나의 객체로 묶어서 관리!
            const [form, setForm] = useState({
                username: '',
                email: ''
            });

            const handleChange = (e) => {
                // e.target에서 name과 value를 미리 추출
                const { name, value } = e.target;

                setForm({
                    ...form,      // 기존 객체 복사
                    [name]: value // 변경이 일어난 input의 name을 key로 삼아 새로운 value로 덮어씀
                });
            };

            const handleSignup = () => {
                alert(`아이디:${username} 이메일: ${email} 회원가입 완료!`);
                setForm({ username: '', email: '' });
            }

            const { username, email } = form;

            return (
                <div>
                    <h2>효율적인 이벤트 처리</h2>
                    <input 
                        type="text"
                        name="username" // state 객체의 key와 동일한 이름
                        placeholder='아이디'
                        value={username}
                        onChange={handleChange}
                    />
                    <input 
                        type="email"
                        name="email"
                        placeholder='이메일'
                        value={email}
                        onChange={handleChange}
                    />
                    <button onClick={handleSignup}>가입하기</button>
                </div>
            )
        }

        function CustomButton({ event, children }) {
            return (
                <button 
                    onClick={event}
                    style={{ backgroundColor: 'black', color: 'white', border: 'none', padding: '10px'}}>
                    {children}
                </button>
            )
        }
        

        function App() {
            const showWelcomeMessage = () => {
                alert('우리 사이트에 오신 것을 환영합니다.');
            }

            return (
                <>
                    <SignupFormBasic />
                    <hr />
                    <SignupFormAdvanced />
                    <div>
                        <h2>Props로 이벤트 함수 전달하기</h2>
                        <CustomButton event={showWelcomeMessage}>
                            환영 메시지 보기    
                        </CustomButton>
                    </div>
                </>
            )
        }

        ReactDOM.createRoot(document.getElementById('root')).render(<App />);
        
    </script>
```
