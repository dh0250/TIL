## 2. 목록 태그

- ul  (Unordered List) : 순서 중요 X
    
    ```html
    		<ul>
            <li>우유</li>
            <li>계란</li>
            <li>사과</li>
        </ul>
    ```
    
- ol (Ordered List) : 순서 중요 O
    
    ```html

            <li>4분 더 끓인다</li>
            <li>맛있게 먹는다!</li>
    ```

## 3. 표 관련 태그

- table 태그 : 전체 표를 감싸는 태그
- caption 태그 : 표 제목이나 설명
- tr 태그 : 행
- th 태그 : 제목 셀.  자동으로 굵게 표시됨
- td 태그 : 열
- thead, tbody

```html
<table>
        <caption>웹 개발 스터디 시간표</caption>
        <thead>
            <!-- tr: Table Row, 표의 한 '행'을 의미 -->
            <tr>
                <!-- th: Table Header, 제목 셀. 자동으로 굵게 표시됨 -->
                <th>시간</th>
                <th>월요일</th>
                <th>수요일</th>
                <th>금요일</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>1교시</td>
                <!-- rowspan="2" : 이 칸은 아래 칸과 합쳐서 '2줄' 차지 -->
                <td rowspan="2">HTML 기초</td>
                <td>JavaScript 기초</td>
                <td>디자인 기초</td>
            </tr>
            <tr>
                <td>2교시</td>
                <!-- colspan-"2": 이 칸은 오른쪽 칸과 합쳐서 '2칸' 차지 -->
                <td colspan="2">JavaScript 기초</td>
            </tr>
        </tbody>
    </table>
```

## 4. 미디어 관련 태그

- img 태그 : 이미지 삽입


- video 태그

```html
<audio src="sample/audio/major.mp3" controls></audio>

<video src="sample/video/video1.mp4" controls></video>
```

## 4. link, form 태그

- a 태그 :  하이퍼링크로 페이지 연결
    - target=”_blank” : 새페이지로 열기

```html
<!-- 다른 웹사이트로 이동 -->
<li><a href="https://www.google.com">구글로 이동</a></li>
<li><a href="https://www.naver.com" target="_blank">네이버로 이동</a></li>

<!-- 같은 폴더 다른 파일로 이동 -->
<li><a href="01_text-and-list.html">1번 파일(글자와 목록)로 이동</a></li>
<li><a href="02_table-and-media.html">2번 파일(표와 미디어)로 이동</a></li>

<!-- 페이지 내부 특정 위치로 이동 -->
<a href="#guestbook">페이지 하단의 방명록으로 바로가기</a>
```

- form 태그
    - action: 데이터를 처리할 서버 주소 / method: 데이터 전송 방식
- fieldset 태그 : 폼 요소를 그룹화하여 테두리를 그려줌
- legend 태그 : 폼 이름
- label for, input id 연결해서 사용성 향상
- input 태그
    - type : radio (하나만 선택) | checkbox (여러 개 선택 가능)

```html
<form action="#" method="post">
        <!-- fieldset: 폼 요소를 그룹화하여 테두리를 그려줌 -->
        <fieldset>
            <legend id="guestbook">방명록 작성</legend>
            <div>
                <!-- label for와 input id를 연결하면 사용성 향상(label을 클릭해도 input에 포커스) -->
                <label for="user-id">아이디:</label> <!--라벨클릭했을때도 인풋박스선택됨-->
                <!-- name은 서버에서 받아올때 데이터에 대한 키값으로 쓰임 -->
                <input id="user-id" name="userId" type="text" placeholder="아이디를 입력하세요">
            </div>
            <div>
                <label for="user-pw">비밀번호:</label>
                <input id="user-pw" name="userPw" type="password">
            </div>
            <div>
                <p>가입 경로 (하나만 선택):</p>
                <input type="radio" id="route-internet" name="route" value="internet">
                <label for="route-internet">인터넷 검색</label>
                <input type="radio" id="route-friend" name="route" value="friend">
                <label for="route-friend">지인 추천</label>
            </div>
            <div>
                <p>관심 분야 (여러 개 선택 가능):</p>
                <input type="checkbox" id="hobby-html" name="hobby" value="html">
                <label for="hobby-html">HTML</label>
                <input type="checkbox" id="hobby-css" name="hobby" value="css">
                <label for="hobby-css">CSS</label>
                <input type="checkbox" id="hobby-js" name="hobby" value="js">
                <label for="hobby-js">JS</label>
            </div>
            <div>
                <label for="memo">하고싶은 말:</label><br>
                <textarea name="memo" id="memo" rows="5" placeholder="자유롭게 작성해주세요"></textarea>
            </div>
            <button type="submit">작성 완료</button>
            <button type="reset">다시 작성</button>
        </fieldset>
    </form>
```

# ✏️CSS

```
				/*
            CSS의 기본 문법
            선택자 {
                속성: 값;
                속성: 값;
            }
        */
        /* 태그 선택자 : HTML 태그 이름을 그대로 사용
        h2 {
            color: red;
        }
        /* 클래스 선택자: 점(.)으로 시작. 여러 요소에 '별명'을 붙여 그룹으로 꾸밀 때 사용 */
        .highlight-text {
            background-color: aquamarine;
        }
        /* 아이디 선택자: 샵(#)으로 시작. 페이지에서 '단 하나뿐인' 요소에 '이름표'를 붙일 때 사용 */
        #blue-text {
            color: blue;
        }

        /* 후손 선택자: 띄어쓰기로 구분. 특정 요소 '안에 있는' 모든 후손을 선택 */
        #lecture-list li {
            font-weight: bold;
        }
        
        /* 자손 선택자: > 기호로 구분. 특정 요소의 '바로 아래 자식'만 선택 */
        body > p {
            font-size: 20px;
        } 
```

- text-align : 텍스트 정렬
- border-radius : 모서리 둥글게
- display: flex; : 부모 요소에 적용하면 flex container가 되고, flex contatiner의 자식 요소는 flex item 됨.
- position: relative (부모요소에 적용) | position: absolute (자식요소에 적용) : 부모요소를 기준으로 위치잡음 (다른요소 상관없이 겹치기 가능)
- grid-template-columns : 사용 가능한 공간을 1 : 1 : 1 비율로 나눔

```
display: grid;
grid-template-columns: 1fr 1fr 1fr;
gap: 10px; /*아이템(요소) 사이의 간격*/
```

- align-items : 교차축(세로) 방향으로 아이템 정렬

## css- interaction

- hover  : 마우스 커서를 올렸을 때의 상태를 선택
- active : 요소를 클릭하고 있는 동안의 상태 선택

```html
			/* 1. Hover: 마우스 커서를 올렸을 때의 상태를 선택 */
        .button:hover {
            background-color: rgb(79, 98, 114);
            color: white;
            /* transporm: 요소의 모양 변형 */
            transform: translateY(-5px);
            box-shadow: 0 5px 10px rgba(0,0,0,0.3);
        }

        /* 2. Active: 요소를 클릭하고 있는 동안의 상태 선택 */
        .button:active {
            transform: translateY(-2px);
        }
```

- text-decoration: none; : 하이퍼링크 밑줄 제거
- transition: all 0.3s ease; : 0.3초동안 부드럽게 움직임
