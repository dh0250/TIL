## 02. variable (변수)

데이터를 메모리에 임시 저장하기 위한 공간과 그에 따른 규칙을 다룸

### ✔ 기본 자료형 (8가지)

```java
// 정수
byte bnum = 1;      // 1byte
short snum = 2;     // 2byte
int inum = 4;       // 4byte (기본)
long lnum = 8L;     // 8byte

// 실수
float fnum = 4.0f;  // 4byte
double dnum = 8.0;  // 8byte (기본)

// 문자 및 논리
char ch = 'A';
boolean isTrue = true;

// 문자열 (참조형)
String str = "Hello";
```

### ✔ 변수 명명 규칙

```java
int age = 20;
int Age = 30;       // 대소문자 구분
int _age, $age;     // 특수문자는 _, $만 허용
int maxAge = 100;   // camelCase 권장 (관례)
// int 1age = 10;   // 숫자로 시작 불가 (에러)
```

## 03. constant (상수)

한 번 저장하면 변경할 수 없는 고정된 값

### ✔ 상수 선언 및 규칙

```java
final int AGE = 20;
// AGE = 30;        // 값 재대입 시 컴파일 에러 발생

// 관례: 대문자 + 언더스코어(_)
final int MAX_VALUE = 100;
final int MIN_VALUE = 0;
```

## 04. overflow (오버플로우)

자료형이 허용하는 최대 범위를 초과했을 때 발생하는 현상

### ✔ 오버플로우 예시

```java
byte num1 = 127;
num1++;
System.out.println(num1); // -128
```

### ✔ 해결 방법

```java
int firstNum = 1000000;
int secondNum = 700000;
long result = (long)firstNum * secondNum;
```

---

## 05. typecasting (형변환)

데이터 타입을 다른 타입으로 변환하는 과정입니다.

### ✔ 자동 형변환

```java
byte b = 10;
int i = b;

long l = 100L;
float f = l;
```

### ✔ 강제 형변환

```java
int inum = 10;
// byte bnum = inum; // 에러
byte bnum = (byte)inum;

double dnum = 3.14;
int i = (int)dnum;  // 소수점 절삭
```

## Section 06. operator (연산자)

데이터를 가공하여 새로운 값을 만들어내는 연산 기호들과 그 활용법을 다룸

---

### ✔ 산술 및 복합 대입 연산자

```java
// 산술 연산자 (+, -, *, /, %)
int a = 10;
int b = 3;
System.out.println(a % b);    // 1 (나머지 연산)

// 복합 대입 연산자
int num = 12;
num += 3;       // num = num + 3; (15)
num %= 5;       // num = num % 5; (0)
```

---

### ✔ 증감 연산자

```java
int firstNum = 20;

// 후위 연산자: 다른 연산 처리 후 증가
int result = firstNum++ * 3;
System.out.println(result);   // 60

// 전위 연산자: 먼저 증가 후 다른 연산 처리
int secondNum = 20;
int result2 = ++secondNum * 3;
System.out.println(result2);  // 63
```

---

### ✔ 비교 연산자

```java
int num1 = 10;
int num2 = 20;

System.out.println(num1 == num2); // false
System.out.println(num1 != num2); // true

// 문자열 비교 (중요)
String str1 = "java";
String str2 = new String("java");

System.out.println(str1 == str2);      // false (주소값 비교)
System.out.println(str1.equals(str2)); // true (값 비교 - 권장)
```

---

### ✔ 논리 연산자 & 삼항 연산자

```java
// && (AND): 모두 참일 때 참
System.out.println(true && false);  // false

// || (OR): 하나라도 참일 때 참
System.out.println(true || false);  // true

// 단축 평가 (Short-circuit)
int n = 10;
boolean result = (false && ++n > 0);
System.out.println(n); // 10 (증가 안됨)

// 삼항 연산자
// (조건식) ? 참일_때_값 : 거짓일_때_값
int score = 85;
String result2 = (score >= 60) ? "합격" : "불합격";

System.out.println(result2); // "합격"
```

## 01. method (메소드)

특정 작업을 수행하기 위해 그룹화된 코드 블록, 코드의 재사용성을 높이고 중복을 방지

---

### ✔ 메소드 호출 및 실행 흐름

```java
// non-static 메소드 호출 방식
// 클래스명 사용할이름 = new 클래스명();
// 사용할이름.메소드명();

Application app = new Application();
app.methodA(); // 메소드 호출 시 해당 블록으로 이동하여 실행 후 복귀
```

---

### ✔ 매개변수와 전달인자

```java
// 전달인자(Argument): 메소드 호출 시 넘겨주는 값
// 매개변수(Parameter): 메소드 선언부에 작성된 변수

app.printAge(20);  // 20이 전달인자

public void printAge(int age) { // int age가 매개변수
    System.out.println("나이: " + age);
}
```

---

### ✔ 반환값 (Return)

```java
// 반환 타입이 void가 아닌 경우 반드시 return문이 필요함
// 반환값은 메소드 호출 위치로 돌아갈 때 가지고 가는 결과값

public String createProfile(String name, int age) {
    return name + "님의 나이는 " + age + "세 입니다.";
}

String message = app.createProfile("코알라", 3); // 결과값을 변수에 저장
```

---

### ✔ static 메소드

```java
// 클래스가 메모리에 로드될 때 생성됨
// 객체 생성 없이 '클래스명.메소드명()'으로 호출 가능

int result = Application3.sumTwoNumbers(10, 20);

// 같은 클래스 내에서는 클래스명 생략 가능
int result2 = sumTwoNumbers(20, 30);
```

---

### ✔ 다른 클래스의 메소드 호출

```java
// 1. non-static: 객체 생성 필수
Calculator calc = new Calculator();
int min = calc.minNumberOf(10, 20);

// 2. static: 클래스명으로 접근
int max = Calculator.maxNumberOf(10, 20);
```

## 02. package_and_import (패키지와 임포트)

클래스들을 체계적으로 관리하기 위한 폴더 구조(패키지)와 다른 패키지의 클래스를 가져오는 방법

---

### ✔ import와 static import

```java
// 다른 패키지의 클래스를 사용할 때 선언
import com.ohgiraffers.section01.method.Calculator;

// static 메소드를 클래스명 없이 사용하고 싶을 때
import static com.ohgiraffers.section01.method.Calculator.maxNumberOf;

Calculator calc = new Calculator(); // import 후 패키지명 생략 가능
int max = maxNumberOf(30, 20);      // 클래스명 없이 호출 가능
```

---

### ✔ Java 표준 API (Math, Random)

```java
// java.lang.Math: 수학 관련 static 메소드 제공 (자동 import)
int min = Math.min(10, 20);
double pi = Math.PI;

// Math.random()을 이용한 난수 발생 (1~10)
int random1 = (int) (Math.random() * 10) + 1;

// java.util.Random: 난수 발생 전용 클래스
import java.util.Random;

Random random = new Random();
int random2 = random.nextInt(10); // 0~9 사이 난수
```
