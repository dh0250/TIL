# 📝class

- 클래스 선언

```jsx
class Student {
    // new 키워드로 객체를 만들 때, 딱 한 번 실행되는 함수
    constructor(name, group) {
        this.name = name;
        this.group = group;
    }

    // 메서드: 이 클래스로 만들어진 모든 객체가 공유하는 기능
    // 이 함수는 객체마다 복사되지 않고, Student.prototype에 저장된다.
    introduce() {
        console.log(`안녕하세요! ${this.group}반 ${this.name}입니다`);
    }
}

const studentA = new Student('홍길동', 'A');
// const student = Student('홍길동', 'A'); // new 키워드 없으면 에러 발생!
const studentB = new Student('유관순', 'B');
```

- 모든 인스턴스가 같은 함수를 씀 → 메모리 절약, 성능 효율

```jsx
console.log(studentA.introduce === studentB.introduce);
```

## ✏️화살표 함수

- 기본 변환 function 키워드를 '=>' 로 바꿈

```jsx
const message = function() { return 'Hello World!'; };
const arrowMsg1 = () => { return 'Hello World!'; };
console.log(arrowMsg1());
```

- 본문이 return 한줄이면, 중괄호{}와 return 동시 생략 가능

```jsx
const arrowMsg2 = () => 'hello simple arrow!';
```

- 매개변수가 하나일 때, 소괄호() 생략 가능

```jsx
const power = function(x) { return x * x; };
const arrowPower = x => x * x;
```

- 매개변수가 없거나, 두 개 이상이면 소괄호() 필수

```jsx
const greet = () => 'hello!';
const sum = (a, b) => a + b;
```

- 객체 리터럴을 반환할 때는 소괄호()로 감싸야 한다.

```jsx
const createUser = (id, name) => ({ id, name });
```

- 고차함수와 함께 사용

```jsx
const numbers = [1, 2, 3, 4, 5];
// const mappedNumbers = numbers.map(function(val) { return val * 10; });
const mappedNumbers = numbers.map( val => val * 10 );
console.log(mappedNumbers);
```

```jsx
        // 일반 함수를 사용한 경우
        // this.titles.forEach(function(title) {
        //     console.log(this.store + ': ' + title); 
        //     foreach가 콜백함수를 부름 이 안에this는 윈도우/글로벌(전역객체)을 가리킴
        //     (window.store = undefined)
        // })

        // 화살표 함수를 사용한 경우
```

- 객체 메서드 = 무조건 일반 함수 / 화살표 함수로 쓰면 this는 전역객체(window)를 가리킴

```jsx
const obj = {
    name: 'han',
    // 객체 내부 메서드를 화살표 함수로 작성할 경우 this는 전역객체(window)를 가리킨다.
    // sayHi: () => {
    //     console.log(this.name);
```

## ✏️iterable

- iterable(이터러블)
    - 순회 가능한  데이터 컬렉션을 위해 통일된 규약(프로토콜)
    - 배열, 문자열, NodeList, Map, Set 등
    - 'for...of' 반복문을 사용

```jsx
const array = ['사과', '바나나', '딸기'];

for (const item of array) {
    console.log(item);
}

const string = '안녕하세요';
for (const char of string) {
    console.log(char);
}
```

- 유사 배열: 인덱스와 length 프로퍼티가 있어서 배열처럼 '생긴 객체’

```jsx
const arrayLike = {
    0: '배열인듯',
    1: '배열아닌',
    2: '유사 배열 객체',
    length: 3
};
```

- Array.from()을 사용하여 '진짜 배열' 로 변환 가능

```jsx
const realArray = Array.from(arrayLike);
console.log(realArray);
```

- 변환과 동시에 map 처럼 활용하기

```jsx
// 1부터 5까지 숫자를 가진 배열 만들기
// 실무에서 초기 데이터를 만들 때 사용됨
const fiveArray = Array.from({ length: 5 }, (value, index) => index + 1);
const fiveArray1 = [0, 0, 0, 0, 0].map((_, i) => i + 1);
```

## 📝Rest Parameter와 Spread 문법

### ✏️Rest Parameter

- 함수의 '매개변수' 자리에 사용하여, 정해지지 않은 개수의 인수들을 하나의 '배열'로 모아서 받음
- Rest 파라미터는 반드시 매개변수 목록의 가장 마지막에 위치

```jsx
function merge(first, ...args) {
    console.log(`첫 번째: ${first}`);
    console.log(`나머지(배열): ${args}`);

    let message = first;
    for (let arg of args) {
        message += arg;
    }

    return message;
}
console.log(merge('안녕하세요'));
// 첫 번째: 안녕하세요
// 나머지(배열): 
// 안녕하세요

console.log(merge('안녕하세요 ', '반갑습니다 ', '저는 판다 입니다.'));
//첫 번째: 안녕하세요
//나머지(배열): 반갑습니다,저는 판다 입니다.
//안녕하세요 반갑습니다 저는 판다 입니다.
```

### ✏️Spread Syntax

- 함수의 '인수' 자리나, 배열/객체 리터럴 안에서 사용하여 배열의 요소들을 개별 값의 목록으로 펼침
- Math.max(10, 20, 30) : 낱개의 숫자들을 인수로 받는다.

```jsx
// Math.max(10, 20, 30) : 낱개의 숫자들을 인수로 받는다.
Math.max(numbers);
console.log(Math.max(numbers)); // NaN 반환

// ...numbers는 [10, 20, 30]을 10, 20, 30으로 펼쳐준다.
console.log(Math.max(...numbers))
```

- 배열 ‘얕은 복사’

```jsx
const arr1Copy = [...arr1];
console.log(arr1Copy);

console.log(arr1 === arr1Copy); // false 주소값 다름

arr1Copy.push(4);  // 복사본 변경
console.log(arr1Copy);  // [ 1, 2, 3, 4 ]
console.log(arr1);      // [ 1, 2, 3 ] -> 원본 유지

// 값 복사(원시값)
let a = 10;
let b = a; // 값 자체를 복사, 주소값 갖고있지 않음

// 주소값 복사
let c = [1, 2];
let d = c;
```

- 객체 리터럴에서 사용

```jsx
const obj1 = { name: '판다', age: 5};
const obj2 = { job: '강사'};

const mergedObject = { ...obj1, ...obj2, location: '서울'}; // 얕은 복사
console.log(mergedObject);

// 객체 '얕은 복사'
const obj1Copy = {...obj1};
obj1Copy.age = 3;     // 복사본 변경
console.log(obj1);    // 원본에 영향 없음
```

## 📝구조 분해 할당

- 배열이나 객체의 속성을 분해하여 그 값을 개별 변수에 쉽게 담을 수 있게 하는 문법
- 기존 방식

```jsx
const nameArr = ['Panda', 'Kwon'];

const firstName = nameArr[0];
const lastName = nameArr[1];
```

- 구조 분해 할당 방식 - 배열은 순서가 기준

```jsx
const [firstName, lastName] = nameArr;
console.log(`성: ${lastName}, 이름: ${firstName}`); //성: Kwon, 이름: Panda
```

- 쉼표 사용해 특정 요소 건너뛰기

```jsx
const [, name] = ['kwon', 'panda'];
console.log(name); // panda
```

- rest 파라미터 사용해 나머지 요소들 새로운 배열로 모으기

```jsx
const [leader, ...members] = ['팀장', '팀원1', '팀원2', '팀원3'];
console.log(leader); //팀장
console.log(members); //[ '팀원1', '팀원2', '팀원3' ]
```

- 할당할 값 없을 때, 기본 값 지정하기

```jsx
const [user1, user2 = '기본값'] = ['사용자1'];
console.log(user1);
console.log(user2);
```

### ✏️객체 구조 분해 할당

- 변수 이름과 같은 키를 찾아 값이 할당됨
- 기존 방식

```jsx
const student = {
    name1: 'panda',
    age: 5,
    major: '영화감상'
}

// 기존 방식 
const name1 = student.name;
const age1 = student.age;
```

- 객체 구조 분해 할당
    - 콜론(:)을 사용해서 'name1'키의 값을 'name1'이 아닌 'studentName'이라는 새 변수명으로 할당

```jsx
const { name1: studentName, age } = student;
console.log(`이름: ${studentName}, 나이: ${age}`); //이름: panda, 나이: 5
```

- 기본 값 설정하기

```jsx
const { name1: sName, job = '학생'} = student;
console.log(job); //학생
```

- rest 파라미터 사용해 나머지 프로퍼티들을 새로운 객체로 모으기

```jsx
const {age: studentAge, ...restInfo} = student;
console.log(restInfo); //{ name1: 'panda', major: '영화감상' }
```

# 📝JavaScript Web

## ✏️DOM (Document Object Model)

### - DOM 요소 선택

- `querySelector`와 `querySelectorAll`

```html
    <ul>
        <li id="apple" class="fruit red">사과</li>
        <li class="fruit">바나나</li>
        <li>딸기</li>
    </ul>
```

```html
		<script>
        // querySelector: CSS 선택자에 맞는 '첫 번째 요소 하나'만 찾아온다.
        const $apple = document.querySelector('#apple');
        $apple.style.fontWeight = 'bold';

        const $firstFruit = document.querySelector('.fruit');
        console.log($firstFruit);

        // querySelectorAll: CSS 선택자에 맞는 '모든 요소'를 NodeList로 찾아온다.
        const $fruit = document.querySelectorAll('.fruit');
        console.log($fruit);

        $fruit.forEach(item => {
            item.style.border = '1px solid blue';
        })
    </script>
```
