## Java 개요 및 개발환경 구축

### 1️⃣ Java 개념

- **Java SE**: 기본 표준 (일반 프로그램 개발용)
- **Java EE**: 웹/서버 개발용 (JSP, Servlet 등)
- **Java ME**: 모바일/임베디드용

---

### 2️⃣ JDK vs JRE

- **JDK**: 개발용 (컴파일 + 실행 도구 포함)
- **JRE**: 실행용 (프로그램 실행만 가능)
    
    → 개발하려면 **JDK 설치 필수**
    

---

### 3️⃣ JDK 종류

- **Oracle JDK**: 유료, 안정적
- **OpenJDK**: 무료, 가장 많이 사용됨
    
    → 보통 **OpenJDK 사용**
    

---

### 4️⃣ JDK 설치 핵심

1. JDK 다운로드 및 압축 해제
2. **JAVA_HOME** 환경변수 설정
3. **Path에 %JAVA_HOME%\bin 추가**
4. `java -version`으로 확인

---

### 📝리터럴(Literal)의 종류

- 정수/실수: 숫자 그대로 작성 (ex: 123, 1.23)
- 문자(char): 홀따옴표(' ')를 사용하며, 단 하나의 문자만 가능 (ex: 'a')
- 문자열(String): 쌍따옴표(" ")를 사용하며, 0개 이상의 문자를 나열 (ex: "안녕하세요", "a")
- 논리(boolean): 참 또는 거짓 상태값 (ex: true, false)

### 📝리터럴의 연산

- 숫자 연산: 사칙연산(+, -, *, /) 및 나머지 연산(%) 가능
- 문자 연산: 문자는 내부적으로 숫자로 취급되어 사칙연산 가능 (ex: 'a' + 'b')
- 문자열 연산: + 연산만 가능하며, 결과는 문자열 합치기 (다른 자료형과 더해도 결과는 문자열)
- 논리 연산: 산술 연산은 불가능하며, 문자열과의 + 연산(결합)만 가능
