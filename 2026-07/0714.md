## 1. 형변환 (Casting) — `02_datatype/09_casting.ipynb`

### 컴퓨터 연산 원칙
- 동일한 자료형 사이에서만 연산이 가능하다.
- 연산 결과 역시 동일한 자료형이다.
  - `int + float` → `float + float`로 형변환 후 연산
  - `int + bool` → `int + int`로 형변환 후 연산

### 암묵적 형변환 (자동)
```python
print(3 + 1.23)      # 4.23 (int -> float)
print(3 + True)      # 4   (bool -> int)
print(3 + int("5"))  # 8
```
- `bool`로 암묵적 형변환되는 규칙
  - 값이 존재하면 `True`
  - 값이 없으면(`0`, `''`, `[]`, `{}`) `False`

### 명시적 형변환
```python
x = int(input('점수입력: '))
print(float('3.5') + 4)  # 7.5
```
- 컨테이너 자료형 간 변환: `list()`, `tuple()`, `set()`, `dict()`
  - `list(tuple)`, `tuple(list)`, `set(list/tuple)` 로 상호 변환 가능
  - `dict([('a', 100), ('b', 200)])` → 리스트 안의 튜플을 딕셔너리로 변환
  - `list(dict)` → 딕셔너리의 key만 리스트로 변환

## 2. 삼항 연산자 — `03_operator/1_operator.ipynb`

```
(참일 때 값) if (조건식) else (거짓일 때 값)
```
- 홀짝 판별, 두 수 중 큰 수/작은 수 판별 등 조건에 따라 값을 즉시 선택할 때 사용
```python
big = a if a > b else b
small = a if a < b else b
```

## 3. 제어문 — `04_control_flow`

### if 조건문 (`01_if_condition`)
- 파이썬은 들여쓰기로 블럭 구성
- `if` : 특정 조건에만 처리 (해당 안 되면 아무 것도 안 함)
- `if ~ else` : 참/거짓 둘 중 하나는 반드시 실행
- `if ~ elif ~ else` : 3가지 이상의 경우의 수, 두 개 이상의 조건식 비교
- 중첩 if문으로 범위 검증 후 세부 분기 처리 가능 (예: 점수 유효성 검사 후 합격/불합격 판정)

### for 반복문 (`02_for_loop`)
- iterable 객체(`str`, `list`, `tuple`, `set`, `dict`) 및 `range` 객체를 반복 대상으로 사용
```python
for ch in name:        # 문자열 순회
for fruit in fruits:   # 리스트 순회
for i in range(0, 10, 2):  # range(start, end, step)
```
- 활용 예: 1~100 홀수 합 구하기, 중첩 for문으로 구구단(2~9단) 출력

### while 반복문 (`03_while_loop`)
- 조건식이 `True`인 동안 반복, `False`가 되면 종료
- 반복 횟수를 미리 알 수 없을 때(사용자 입력 등) 주로 사용
- 증감변수로 반복 종료 시점 제어
```python
i = 0
while i < 3:
    print('🍉')
    i += 1
```
- `while True:` + `break`로 무한 루프를 강제 종료하는 패턴

### 내포 comprehension (`04_comprehension`)
- 한 줄짜리 반복문으로 리스트/딕셔너리를 생성하는 문법
```python
nums = [n + 100 for n in range(1, 101)]          # list comprehension
nums_ = [n for n in range(1, 101) if n % 2 == 0] # 조건 포함
target = {k: v for k, v in zip(students, scores)} # dict comprehension
```

## 4. 함수 — `05_function`

### 함수 기본 (`01_function`)
- 절차적/반복적 실행 코드를 가진 자료형, 선언 후 호출해서 사용
- 매개변수(parameter, 선언부 변수) vs 매개인자(argument, 호출부 전달 값)
- `return` 명시 안 하면 함수는 `None` 반환

### 호출 방식
- 위치 매개인자: 선언 순서대로 대입
- 키워드 매개인자: `name=value` 형태로 순서 무관하게 대입
- 기본값 지정 매개변수는 호출 시 생략 가능하나, 기본값 없는 매개변수 뒤에 올 수 없음

### packing / unpacking 연산자
- `*args` : n개의 위치 매개인자를 튜플로 받음
- `**kwargs` : n개의 키워드 매개인자를 딕셔너리로 받음
- packing 매개변수는 일반 매개변수보다 오른쪽에 선언
```python
def sample(a, b, c=99, *args, **kwargs):
    print(a, b, c, args, kwargs)
```
- unpacking(`*`, `**`)은 값 전달 위치에서 컨테이너 요소를 개별 값으로 풀어서 전달
```python
test(*names)  # 리스트 요소를 개별 인자로 전달
```

### 1급 시민 객체로서의 함수
- 함수는 변수에 대입 가능, 다른 함수의 인자/반환값으로 사용 가능
- 함수 내부에서 함수를 정의하고 반환하는 클로저 형태 가능

### lambda 함수 (`02_lambda`)
- 한 줄로 작성하는 익명 함수 표현식: `lambda 매개변수: 리턴값`
```python
f = lambda x: x ** 2
add = lambda a, b=3: a + b       # 기본값 지정 가능
adder = lambda *args: sum(args)  # packing 사용 가능
```
- comprehension 안에서 즉시 실행하는 형태로도 사용 가능

## 5. 내장(built-in) 모듈/함수 — `06_api/01_builtins.ipynb`

- `dir(__builtins__)` : import 없이 사용 가능한 내장 함수/속성 목록 확인

### map
- iterable의 각 요소에 함수를 적용한 결과를 map 객체로 반환
```python
lens = [*map(lambda x: len(x), texts)]
list(map(lambda mid, final: (mid + final) / 2, mid_terms, final_terms))  # 여러 iterable 동시 사용
```

### filter
- iterable에 함수를 적용해 `True`인 값만 필터링
```python
evens = list(filter(lambda x: x % 2 == 0, nums))
shins = list(filter(lambda n: n.startswith('신'), names))
```

### reduce (`functools`)
- iterable 요소들을 지정한 연산으로 누적하여 단일 값을 반환
- 매개변수: `함수(누적값, 현재값)`, iterable 객체, 초기값(생략 시 첫 요소가 초기값)
```python
from functools import reduce
total = reduce(lambda acc, current: acc + current, nums, 0)
max_num = reduce(lambda a, b: a if a > b else b, numbers)
```
