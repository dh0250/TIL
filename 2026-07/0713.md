## 1. Python 개요

**Python이란**
1991년 귀도 반 로썸이 ABC 언어에서 영감을 받아 만든 프로그래밍 언어로, 속도보다는 생산성에 중점을 둔다.

**Python의 철학**
가독성, 명시성, 단순함을 중시하며(Zen of Python), 오픈소스 정신에 따라 커뮤니티 참여를 독려한다.

**사용 이유**
- 배우기 쉽고 다양한 시스템과 호환됨
- 웹 개발, 데이터 분석, AI/머신러닝, 자동화 등 폭넓게 활용됨
- 직관적인 문법으로 복잡한 문제도 간단히 해결 가능
- 현재 가장 인기 있는 언어 중 하나

**특징**
- 오픈소스, 무료
- 인터프리터 방식의 객체지향 언어, 동적 타이핑
- 스크립트/글루 언어로도 적합
- 모듈/패키지 재사용 용이

**장단점**
- 장점: 강력함, 빠름, 높은 호환성, 배우기 쉬움
- 단점: Python을 먼저 배우면 다른 언어 학습 시 진입장벽을 느낄 수 있음

## 2. 개발환경 구축

**conda-forge**
- Python 패키지/가상환경 관리 도구 (Anaconda와 달리 커뮤니티 기반 오픈소스)
- 설치 → Miniforge 프롬프트 실행 → `conda create -n [환경명] python=[버전]`으로 가상환경 생성 → `conda info --envs`로 확인
- 삭제: `conda remove --name [환경명] --all`

**Jupyter Notebook**
- 대화형 컴퓨팅을 지원하는 무료 웹 기반 도구
- 설치 순서: 가상환경 활성화(`conda activate [환경명]`) → `pip install jupyter notebook` → `pip install ipykernel` → `python -m ipykernel install --user --name [환경명] --display-name [표시명]`
- base 환경이 아닌 별도 가상환경에 설치 권장

**VSCode**
- 다양한 언어와 확장 기능을 지원하는 코드 편집기, Git 연동 및 통합 터미널 제공
- 설치 후 Python/Jupyter 확장 설치 → 명령어 팔레트에서 인터프리터 선택 → `.ipynb` 파일 생성 → 커널 선택으로 가상환경 연결

# 파이썬 변수와 자료형 정리

## 1. 변수 (Variable)

**값 하나**를 보관하기 위해 이름을 붙인 메모리 공간이다.

```python
a = 10
a = 20  # 재대입 시 이전 값을 덮어씀
print(a)  # 20
```

### 변수명 작성 규칙

1. 대소문자를 구분한다 (`A`와 `a`는 다른 변수).
2. snake_case를 사용한다 (단어 사이를 `_`로 연결).
3. 한글 변수명도 가능하지만 인코딩 문제를 유발할 수 있어 지양한다.
4. `_`를 제외한 특수문자는 사용할 수 없고, 숫자로 시작할 수 없다.
5. 파이썬 예약어(`if`, `else`, `for`, `while` ...)는 변수명으로 사용할 수 없다.
6. 짧고 의미 없는 `a`, `b`, `c` 대신 **직관적인 변수명**을 사용한다.

### 주석

```python
# 한 줄 주석

'''
여러 줄 주석
'''

"""
여러 줄 주석
"""
```

## 2. 자료형 (Data Type)

파이썬의 자료형은 **기본 자료형**과 **컨테이너 자료형**으로 나뉜다.

### 기본 자료형 (Primitive Types)

| 자료형 | 설명 | 예시 |
| --- | --- | --- |
| `int` | 정수형 | `10`, `-3`, `0` |
| `float` | 실수형 | `3.14`, `-0.1` |
| `bool` | 불리언 | `True`, `False` |
| `str` | 문자열 | `"hello"`, `'파이썬'` |
| `NoneType` | 값이 없음을 의미 | `None` |

### 컨테이너 자료형 (Container Types)

| 자료형 | 설명 | 예시 |
| --- | --- | --- |
| `list` | 순서 있고 변경 가능 | `[1, 2, 3]` |
| `tuple` | 순서 있고 변경 불가능 | `(1, 2, 3)` |
| `set` | 중복 불허, 순서 없음 | `{1, 2, 3}` |
| `dict` | 키-값 쌍의 집합 | `{"name": "Tom"}` |

### 시퀀스형 vs 이터러블형

- **시퀀스형**(`list`, `tuple`, `str`, `range`)은 순서가 있어 인덱싱·슬라이싱이 가능하다.
- **이터러블형**은 `for`문으로 순회 가능한 모든 객체(`list`, `set`, `dict`, `str`, 파일객체 등)를 뜻하며, `iter()`로 이터레이터를 만들 수 있다.
- 즉 모든 시퀀스형은 이터러블이지만, 모든 이터러블이 시퀀스는 아니다(`set`, `dict`는 인덱싱 불가).

## 3. 숫자형 (int, float)

```python
m, n = 10, 3

print(m + n)   # 13
print(m - n)   # 7
print(m * n)   # 30
print(m / n)   # 3.3333333333333335 (나눗셈은 항상 float)
print(m // n)  # 3  몫
print(m % n)   # 1  나머지
print(m ** 2)  # 100 제곱
```

- `round(n, 자리수)`: 반올림
- `math.ceil(n)`: 올림, `math.floor(n)`: 내림 (`math` 모듈 필요)

---

## 4. 논리형 (bool)

`True`(1) / `False`(0) 두 가지 값만 가진다.

```python
print(10 > 4)   # True
print(10 <= 4)  # False
print(not True)  # False

print(True and False)  # False, 둘 다 참이어야 True
print(True or False)   # True, 하나만 참이어도 True
```

조건이 여러 개 섞이면 괄호 `()`로 우선순위를 명확히 하는 것이 좋다.

## 5. 문자열 (str)

```python
s = "문자열"          # 작은따옴표/큰따옴표 동일
s = """여러줄
문자열"""             # 삼중따옴표는 여러 줄 지원

print("가" + "나")    # 연결
print("가" * 3)       # 반복: 가가가
```

### 인덱싱 / 슬라이싱 (시퀀스 공통)

```python
text = "hello world"
text[0]      # 'h'
text[-1]     # 'd' (역순 인덱스)
text[1:4]    # 'ell'  [start:end)
text[::-1]   # 'dlrow olleh' (뒤집기)
```

### 주요 str 메서드

| 메서드 | 설명 |
| --- | --- |
| `len(s)` | 길이 |
| `'x' in s` | 포함 여부 (대소문자 구분) |
| `s.replace(a, b)` | 치환 (원본 불변, 새 문자열 반환) |
| `s.strip()` | 양끝 공백 제거 |
| `s.upper()/lower()` | 대/소문자 변환 |
| `s.capitalize()/title()/swapcase()` | 대소문자 스타일 변환 |
| `s.split(구분자)` | 문자열을 리스트로 분리 |

### f-string

```python
name, age = '홍길동', 33
print(f"저는 {name}이고 {age}살입니다.")
```

## 6. list

순서가 있고 **중복을 허용**하며 **변경 가능(mutable)**한 자료형.

```python
nums = [1, 2, 3]
nums.append(4)      # 맨 뒤 추가, in-place
nums.insert(1, 1.5) # 특정 위치에 삽입

nums[1:3] = [7, 8, 9]  # 슬라이스로 구간 교체(제거 후 삽입)
```

- `list.index(v)`, `list.count(v)`: 위치/개수 찾기
- `list.sort(reverse=True, key=len)`: 원본 정렬(반환값 `None`)
- `sorted(list, ...)`: 새 리스트 반환(원본 유지)
- `enumerate(lst)`: 인덱스와 값을 함께 순회

---

## 7. tuple

순서가 있지만 **변경 불가능(immutable)**한 자료형.

```python
t = (10, 20)     # 괄호 생략 가능: t = 10, 20
single = (10,)   # 요소 1개는 콤마 필수

x, y = 10, 20
x, y = y, x      # 튜플 언패킹으로 값 교환
```

튜플 자체는 불변이지만, 내부에 담긴 `list` 같은 가변 객체는 수정할 수 있다.
```python
hong = ('홍길동', 33, [100, 89, 55])
hong[2][2] = 70  # 가능 (list 내부 요소 변경)
```

## 8. dict

key-value 쌍을 저장하며, key로 값을 찾으므로 인덱싱/슬라이싱은 지원하지 않는다.

- key: 중복 불허, `int/str/tuple` 등 불변 타입만 가능
- value: 어떤 타입이든 가능 (list, dict, 함수 등)

```python
user = {"name": "이순신", "hobby": ["수영"], "pet": {"name": "댕댕이"}}
print(user['pet']['name'])  # 댕댕이

user.get('nickname', '없음')  # 키가 없으면 기본값 반환
```

- `dict.keys() / values() / items()`: 각각 키, 값, (키,값) 쌍을 순회
- `zip(a, b, c)`: 여러 리스트를 묶어서 동시에 순회, `zip(*data)`로 언패킹(역연산) 가능

---

## 9. set

**중복을 허용하지 않는** 자료형. 순서가 없어 인덱싱 불가.

```python
s = {1, 2, 2, 3}     # {1, 2, 3} 자동 중복 제거
s.add(4)
s.remove(2)          # 없으면 KeyError
s.discard(100)       # 없어도 에러 없음
```

### 집합 연산

```python
a = {1,2,3,4,5}
b = {3,4,5,6,7}

a.union(b)                # 합집합
a.intersection(b)         # 교집합
a.difference(b)           # 차집합 (a-b)
a.symmetric_difference(b) # 대칭차집합 (합집합 - 교집합)
```

## 한 줄 정리

> 변수는 값을 담는 이름 붙은 공간이며, 파이썬 자료형은 크게 int/float/bool/str 같은 기본 자료형과 list/tuple/dict/set 같은 컨테이너 자료형으로 나뉜다. list는 가변·순서 있음, tuple은 불변·순서 있음, dict는 key-value 매핑, set은 중복 불허·순서 없음이 핵심 차이다.
