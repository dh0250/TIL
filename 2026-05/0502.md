## 03. Abstraction & DTO

객체지향 프로그래밍에서 실세계의 사물을 추상화하여 클래스로 정의하고, 데이터를 효율적으로 전달하기 위한 구조

---

### ✔ 추상화 (Abstraction)

유연한 설계를 위해 실세계의 객체를 프로그램에 필요한 속성과 행위만 추출하여 클래스로 만드는 과정

```java
// 추상화된 클래스 예시 (Car.java)
public class Car {
    private boolean isOn; // 상태(필드)
    private int speed;

    public void startUp() { /* 행위(메소드) */ }
    public void go() { /* 가속 기능 */ }
    public void stop() { /* 제동 기능 */ }
}
```

---

### ✔ DTO (Data Transfer Object)

계층 간 데이터를 전달하기 위해 사용하는 객체로, 필드 보호(캡슐화)와 표준화된 접근 방식을 제공

**작성 규칙**

1. 모든 필드는 private으로 선언 (캡슐화)
2. 기본 생성자와 매개변수 있는 생성자 작성
3. 모든 필드에 대한 Getter와 Setter 제공

```java
public class MemberDTO {
    private String name;
    private int age;

    public MemberDTO() {} // 기본 생성자

    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
}
```

---

## 04. Constructor (생성자)

new 연산자를 통해 객체가 메모리(Heap)에 생성될 때, 가장 먼저 호출되는 객체 초기화 메소드

---

### ✔ 생성자 작성 규칙

* 이름이 클래스명과 반드시 동일해야 함
* 반환 타입(void 등)을 작성하지 않음

---

### ✔ 기본 생성자와 매개변수 생성자

```java
public class User {
    private String id;
    private String pwd;
    private String name;

    // 기본 생성자
    public User() {}

    // 매개변수 생성자
    public User(String id, String pwd) {
        this.id = id;
        this.pwd = pwd;
    }

    // this() 사용
    public User(String id, String pwd, String name) {
        this(id, pwd);
        this.name = name;
    }
}
```

## 05. Overloading & Parameter

메소드의 확장성을 높이는 오버로딩 규칙과 파라미터 전달 방식

---

### ✔ 메소드 오버로딩 (Overloading)

동일한 이름의 메소드를 매개변수의 타입, 개수, 순서를 다르게 하여 여러 개 정의하는 기술

* 성립 조건: 매개변수 시그니처가 달라야 함
* 불성립 조건: 반환 타입이나 접근 제한자만 다른 경우

```java
public class Calculator {
    public void test() {}
    public void test(int num) {}             
    public void test(String str) {}          
    public void test(int num, String str) {} 
}
```

---

### ✔ 메소드 파라미터 (Parameter)

1. 기본 자료형 → 값이 복사되어 전달 (원본 변경 불가)
2. 참조 자료형 → 주소값이 복사되어 전달 (원본 변경 가능)
3. 가변 인자 → 파라미터 개수가 가변적일 때 사용 (...)

```java
// 가변 인자 사용 예시
public void orderPizza(String name, String... toppings) {
    System.out.println(name + "님의 토핑: " + Arrays.toString(toppings));
}

// 호출
orderPizza("홍길동", "치즈", "페퍼로니", "버섯");
```
